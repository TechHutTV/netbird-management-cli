Fetch the NetBird API documentation from https://docs.netbird.io/api

The NetBird API has the following main resources:
- **Accounts** - Account information and management
- **Users** - User account management
- **Tokens** - Personal access token management
- **Peers** - Network peer management
- **Setup Keys** - Device onboarding key management
- **Groups** - User and device organization
- **Policies** - Network access rule definitions
- **Posture-Checks** - Device compliance validation
- **Networks** - Network infrastructure management
- **DNS** - DNS settings and nameserver groups
- **Events** - Audit logs and activity records

Use the WebFetch tool to get documentation for the specific resource the user is asking about.

Example: If asked about peer endpoints, fetch https://docs.netbird.io/api and extract information about peer-related endpoints.
